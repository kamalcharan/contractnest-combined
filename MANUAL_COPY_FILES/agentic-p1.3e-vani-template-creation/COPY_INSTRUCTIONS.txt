WHAT THIS DOES:
  - Templates hub gets a "Draft with VaNi" button (VaNi subscribers only —
    same entitlement gate as the contracts hub).
  - Same canvas, SHORTER pipeline: describe the offering ("1 year HVAC AMC
    with quarterly PM visits and sign-off acceptance") → AI parse → catalog
    scan → AI block selection → assemble. NO buyer step, NO dates — a
    template has no counterparty.
  - Duplicate guard: if a signed-off template already matches the request,
    an info card says so ("Similar template already exists: X") and
    composing continues — compare before signing off.
  - Review shows the standard contract view (buyer blank, as in the template
    wizard); footer = template name input (prefilled) + "Save Template" →
    saves a DRAFT template with the full wizard_state + defaults payload
    (identical shape to wizard-built templates, so editing round-trips
    through the template wizard and the VaNi match tier can consume it
    once signed off).
  - Sign-off stays on the Templates hub — VaNi never activates a template.
