  4th parameter. Rejects the claim outright (code: ENVIRONMENT_MISMATCH)

  workspace, but you are currently in Test mode...").

  (already computed in the outer request scope) as p_is_live.
- No UI change needed — the shared `api` axios instance already attaches
  x-environment globally via an interceptor, so the claim call already
  carries the correct current Test/Live toggle state.
