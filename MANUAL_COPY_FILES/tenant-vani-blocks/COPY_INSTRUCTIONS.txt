Full-file copies — Copy-Item each over the original (1 NEW, 1 REPLACE).
Frontend only. No DB / API / edge changes. Recommendation is a local,
deterministic heuristic in this slice (asset relevance) — a later batch can
swap it for an API/edge call without changing the component surface.

WHAT THIS IS
  The tenant agent. On the contract wizard's "Add Service Blocks" step, a
  "Suggest blocks with VaNi" button opens a right drawer that reads the
  TENANT'S OWN catalog blocks (same source as the block library), surfaces the
  ones relevant to the contract's assets (services + spares), lets the tenant
  accept/reject, and adds the accepted ones straight into the wizard's
  selectedBlocks. It never invents blocks — it only picks from what the tenant
  already has. This is the "recipe = pick your blocks" model, accelerated.

SUBMODULES AFFECTED: contractnest-ui  (ONLY)
