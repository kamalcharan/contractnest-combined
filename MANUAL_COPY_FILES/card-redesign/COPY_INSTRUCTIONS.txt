WHAT CHANGED

Redesigned TemplateCard component to match Catalog Studio templates-list.tsx card UX:

GRID VIEW:
- Rounded-xl card with hover lift animation (hover:-translate-y-1, hover:shadow-lg)
- Header section (border-bottom): Lucide LayoutTemplate icon box + Global badge + Status badge
- Title + 2-line clamped description
- Contract Type + Complexity inline pill badges
- Stats: 2-column grid (Usage count + Rating) with subtle background
- Meta row: Duration + Blocks count
- Footer (border-top): Full-width primary action button + Preview icon + 3-dot dropdown
- Dropdown menu: Preview, Duplicate, Edit, Delete (admin only)
- Theme-aware inline styles (dark/light mode)
- Selected state: brand-colored ring

LIST/COMPACT VIEW:
- Horizontal layout with icon, title, badges, stats, actions
- Same theme-aware styling
- Inline action button + 3-dot dropdown

BACKWARD COMPATIBLE:
- Same props interface (TemplateCardProps)
- Same context system (TemplateCardContext)
- Works with both global-templates.tsx and service-contracts/templates/index.tsx
