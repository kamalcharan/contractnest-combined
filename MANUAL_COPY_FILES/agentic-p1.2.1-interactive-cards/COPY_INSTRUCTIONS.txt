     needs you (not step 1); every "needs you" chip on the readiness card is
     a deep link into its exact wizard step (new vaniInitialStepId prop).

     start / currency / billing); change them via ← Back → Preferences card.
  6. TENANT DEFAULT CURRENCY — contract currency now comes from your tenant
     default (getDefaultCurrency), changeable on the Preferences card; if any
     block is priced in a different currency VaNi flags it (never silently
     converts).
