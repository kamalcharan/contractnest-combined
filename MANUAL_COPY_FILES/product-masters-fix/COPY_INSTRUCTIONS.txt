PRODUCT MASTERS FIX - COPY INSTRUCTIONS
═══════════════════════════════════════════════════

ISSUE FIXED:
- Product Masters page not loading data
- Root cause: transformForFrontend() was applying transformMasterData() to category list responses
- transformMasterData() expects category DETAILS fields (sub_cat_name, detail_name, etc.)
- But category LIST responses have different fields (category_name, etc.)
- This was corrupting the data structure, causing empty/undefined values
