═══════════════════════════════════════════════════
WHAT THIS WIREFRAME COVERS
═══════════════════════════════════════════════════

Premium Contract Wizard with Floating Action Island (6 Steps):

- Step 0: Path Selection
  * "From Template" - Uses pre-built template with blocks
  * "Create from Scratch" - Build contract ad-hoc

- Step 1: Buyer Selection
  * Recent buyers quick-select
  * Search buyers functionality
  * Quick add new buyer

- Step 2: Acceptance Options
  * Accept on Payment
  * Email/OTP Signoff
  * Auto Accept

- Step 3: Base Details
  * Contract name
  * Duration (months)
  * Currency selection
  * Description

- Step 4: Block Assembly
  * Select service blocks from library
  * Configure: Cycle (days), Quantity, Price
  * Unlimited toggle
  * Real-time price calculation

- Step 5: Review & Send
  * Paper Canvas style review
  * All contract details visible
  * Send channel selection (WhatsApp/Email/Link)
