COMPONENT DESCRIPTIONS
═══════════════════════════════════════════════════════════════════════════════

BlockCardSelectable.tsx
- Compact block card for library display
- Shows icon, name, description, category, price
- +Add button with selected state

BlockCardConfigurable.tsx
- Block card with inline accordion settings
- Drag handle, quantity controls, cycle selector
- Pro-rate, auto-renew, custom price options
- Uses same pattern as template.tsx drag-drop

BlockLibraryMini.tsx
- Compact library panel with collapsible categories
- Search functionality
- Shows block counts per category

ContractPreviewPanel.tsx
- Live contract preview (Column 3)
- Tenant info from useTenantProfile()
- Customer info from Step 2
- Contract details from Step 3
- Blocks summary with totals

ServiceBlocksStep.tsx
- Main 3-column layout component
- Drag-drop reordering (same as template.tsx)
- Integrates all components
