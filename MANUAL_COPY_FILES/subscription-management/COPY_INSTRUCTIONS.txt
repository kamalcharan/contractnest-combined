WHAT'S INCLUDED
═══════════════════════════════════════════════════════════════════════════════

This package includes the complete UI layer for Subscription Management:

PHASE 1: ADMIN UI
─────────────────────────────────
1. Admin Dashboard (/admin/subscription-management)
   - Platform Pulse stats with animated counters
   - Tenant list with filters (status, subscription, type, industry)
   - Grid and List view modes
   - Tenant detail drawer
   - Multi-step delete confirmation flow

PHASE 2: TENANT OWNER UI
─────────────────────────────────
2. Close Account Page (/settings/business-profile/close-account)
   - Empathetic "We're sorry to see you go" messaging
   - Optional feedback collection
   - Data summary preview
   - Multi-step delete confirmation

3. Close Account Section (embeddable component)
   - Can be added to Business Profile page
   - Compact version of close account flow

SHARED COMPONENTS
─────────────────────────────────
- Badges: SubscriptionStatusBadge, TenantStatusBadge, TenantTypeBadge
- Cards: StatCard, TenantCard
- Data Viz: AnimatedCounter, DataProgressBar, DataCategoryCard
- Filters: TenantFilters
- Modals: TenantDetailDrawer, DeleteConfirmationFlow
- Owner: CloseAccountSection
