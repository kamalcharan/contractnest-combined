  query refetch to swap in the view page. For a NEW profile, existingProfile

  section). So it looked like "no view page".

      view page renders with no blank-wizard flash.

      the view page.
    • Then invalidateQueries to reconcile with the canonical server row.
    • Toast now says: "Profile saved! Add semantic clusters below."
  The view page already had the full clusters UI (generate / add / edit /
  delete / save) on the right — same role as the Groups dashboard. No new
  view page needed; you just now actually land on it.

  NOTE: the "basic save" does not generate the embedding (that happens via the
  clusters/generate path), so the "Vector Indexed" badge appears only after
