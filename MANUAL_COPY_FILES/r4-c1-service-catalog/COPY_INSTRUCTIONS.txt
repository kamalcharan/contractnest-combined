What it does:
  - Adds "Service Catalog" sidebar tab to Business Profile
  - Shows All | Equipment | Facilities filter tabs with counts
  - Type badges on each row (Equipment blue, Facility purple, Consumable yellow)
  - Stats bar: X equipment, Y facilities, Z with Knowledge Tree, N pending
  - KT coverage status per item (green check or yellow warning)
  - "View KT" link navigates to KT Builder detail page
  - Search bar filters by name, sub_category, description
  - Info banner about future seeding capability
