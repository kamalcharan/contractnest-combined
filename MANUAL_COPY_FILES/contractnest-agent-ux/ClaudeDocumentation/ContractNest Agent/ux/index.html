ContractNest Agent

A 100% agentic redesign. Not "a chatbot bolted on" — the app has agency. The system arrives with the work done, hand-holds you to the outcome, and meters what the agent does for you.

🧭 Layer 1 · Copilot Agentic UI

You drive — the UI hand-holds. Zero blank states, pre-filled drafts, next-best-action everywhere. Included in the plan. "I'll do it myself, but I'm never staring at an empty form."

🤖 Layer 2 · VaNi Autopilot

VaNi drives — you approve. Acts on incoming WhatsApp / email, creates contracts, schedules service, invoices & follows up. Human-in-loop. Metered as top-up credits.

↔

One engine, two initiators — the dial is also the revenue dial

Same tools, same proposed-action card, same guardrails (Permission · Risk · Affordability), same audit trail. "Do it myself" is free; "VaNi, you handle it" burns credits.
