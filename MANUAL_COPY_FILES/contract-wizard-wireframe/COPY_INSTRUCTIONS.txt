WHAT THIS WIREFRAME COVERS
═══════════════════════════════════════════════════

Contract Creation Journey (6 Steps):
- Step 0: Path Selection (Template vs Ad-hoc)
- Step 1: Buyer Selection (search + quick add)
- Step 2: Acceptance Options (payment/signoff/auto)
- Step 3: Base Details (name, duration, currency)
- Step 4: Block Assembly (add/configure blocks)
- Step 5: Review & Send (WhatsApp/Email/Link)

Features Demonstrated:
- Full-screen wizard overlay
- Split-screen layout (content + live preview)
- Visual progress pills (not boring numbered steps)
- Block settings drawer (cycle, qty, unlimited toggle)
- Live contract preview (buyer POV)
- Send channel selection
- Animations and micro-interactions
