RESEED / CLEANUP PATTERN (for future use)
─────────────────────────────────────────────────────
To wipe and reseed sample contacts for a tenant:

  -- Step 1: Delete seed contacts
  DELETE FROM t_contacts WHERE tenant_id = '<uuid>' AND is_seed = true;

  -- Step 2: Reseed via API
  POST /api/seeds/tenant/sample-contacts
  Headers: Authorization: Bearer <token>, x-tenant-id: <uuid>
  Body: { "industryId": "healthcare" }
