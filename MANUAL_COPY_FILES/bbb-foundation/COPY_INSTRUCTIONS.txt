WHAT THIS BATCH CONTAINS (4 features):
1. Onboarding LOV setup — new VaNi step 'lov-setup' between equipment-confirm
   and done. Tenant creation now seeds Roles (Owner/Admin/Member) + Tags
   (Lead/Guest/VIP); the step lets users review/add values, fully driven by
   DEFAULT_LOV_SEED (add a category to the config = it appears in the step).
2. Contacts: Tags in Quick Add drawer (chip toggles from the Tags LOV).
3. Contacts: Industries multi-select (sub-industries, same master data as
   business-profile) on the full contact form; stored as jsonb on t_contacts.
   PLUS: Contact Persons section now available for INDIVIDUAL contacts as
   "Alternative Contact Person" (stand-in / replacement attendance).
4. Import wizard — /contacts/import (entry: "Import" button on Contacts page).
   Upload .xlsx/.csv -> map columns -> review (incl. Category->Industry
   mapping) -> batched import (25/request) -> per-row results report.
   New API endpoint: POST /api/contacts/import (10 batches / 15 min).
